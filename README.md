# 02_04 Challenge: Develop a Container Image Workflow

## Challenge tasks

To complete this challenge, you will:

- Create a new GitHub repository for a Python project that includes a Dockerfile
- Add the exercise files to the repo
- Configure a Python application workflow using a starter template
- Update the repo to use a workflow_call trigger
- Update Python versions and GitHub Action versions
- Configure a Publish Docker Container workflow using a starter template
- Add the integration job to container workflow
- Add permissions for the integration job to read the repo contents and write checks if needed
- Update the build job to depend on the successful completion of the integration job
- Run the workflow and confirm all jobs finish without errors
- Confirm the container image was created


- Trigger and verify a successful CI run
